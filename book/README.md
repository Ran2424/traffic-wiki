# Book collection

本目录收录 4 本交通领域书籍的 MinerU Markdown。每本书的目录包含：

- 一个按正文顺序合并的 Markdown；
- `source-parts/`：从 `/Users/ran/Downloads` 复制的原始分卷，统一命名为 `part-01.md`、`part-02.md` 等。

合并顺序依据 MinerU 文件 ID、相邻卷首尾句和章节连续性共同确认。`source-parts/` 保持原始字节不变；合并版在顺序拼接后整理了 Markdown 标题层级，并补回少量可由原书目录明确确认的章、节标题。正文 OCR 文本和远程图片链接未作清洗或改写。
